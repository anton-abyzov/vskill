# Backend Development Plugin

**Version**: 1.1.0
**Author**: vskill Contributors
**License**: MIT

## Description

Full-stack backend development across 7 languages and frameworks: Node.js, .NET, Python, Go, Java/Spring, Rust, and GraphQL. Database optimization, API design, and enterprise patterns.

## Skills

| Skill | Description |
|-------|-------------|
| dotnet-backend | .NET/C# backend with ASP.NET Core 8+, Entity Framework Core, and enterprise patterns |
| nodejs-backend | Node.js/TypeScript backend with Express, Fastify, NestJS, and APIs |
| python-backend | Python backend with FastAPI, Django, Flask, and async patterns |
| database-optimizer | Database optimization for query performance, indexing, and schema design |
| go-backend | Go/Golang backend with Gin, Echo, Chi, sqlx, and cloud-native patterns |
| java-spring | Java/Spring Boot 3.x backend with JPA, Security 6, WebFlux, and Virtual Threads |
| rust-backend | Rust backend with Axum, Actix-web, SQLx, Tokio, and zero-cost abstractions |
| graphql | GraphQL API design across Node.js, Python, .NET, and Java with Federation support |

## Installation

```bash
vskill add --repo anton-abyzov/vskill --plugin backend
```

## Requirements

- vskill CLI installed
