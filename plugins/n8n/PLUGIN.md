# n8n Kafka Workflows

**Version**: 1.0.0
**Author**: vskill Contributors
**License**: MIT

## Description

Event-driven workflow automation with Apache Kafka integration. Master Kafka trigger nodes, producer nodes, event-driven workflow patterns, error handling, retries, and dead letter queue management for no-code/low-code event processing.

## Skills

| Skill | Description |
|-------|-------------|
| n8n-kafka-workflows | Expert knowledge of integrating Apache Kafka with n8n workflow automation platform for event-driven processing patterns |

## Commands

| Command | Description |
|---------|-------------|
| /n8n:workflow-template | Generate n8n workflow JSON template with Kafka integration patterns (enrichment, fan-out, retry+DLQ, CDC) |

## Installation

```bash
vskill add --repo anton-abyzov/vskill --plugin n8n
```

## Requirements

- vskill CLI installed
- n8n instance (self-hosted or cloud)
- Apache Kafka broker (local or cloud-hosted)
