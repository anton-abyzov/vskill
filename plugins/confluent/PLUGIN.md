# Confluent Stream Processing Plugin

**Version**: 1.0.0
**Author**: vskill Contributors
**License**: MIT

## Description

Expert knowledge and guidance for Confluent Kafka ecosystem including Kafka Connect, ksqlDB, and Schema Registry. Build real-time data pipelines, stream processing applications, and manage event schemas at scale. Covers CDC patterns, single message transforms, windowing, aggregations, and schema evolution strategies.

## Skills

| Skill | Description |
|-------|-------------|
| confluent-kafka-connect | Kafka Connect integration expert for source and sink connectors, JDBC, Elasticsearch, S3, Debezium CDC, and SMT patterns |
| confluent-ksqldb | ksqlDB stream processing expert for SQL-like queries on Kafka topics, joins, aggregations, windowing, and materialized views |
| confluent-schema-registry | Schema Registry expert for Avro, Protobuf, and JSON Schema management with evolution strategies and compatibility modes |

## Commands

| Command | Description |
|---------|-------------|
| (Integrated skills - no separate commands) | Skills activate automatically based on context and keywords |

## Installation

```bash
vskill add --repo anton-abyzov/vskill --plugin confluent
```

## Requirements

- vskill CLI installed
- Confluent Kafka cluster (Cloud or self-hosted)
