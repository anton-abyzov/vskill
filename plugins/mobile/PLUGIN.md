# Mobile Plugin

**Version**: 1.1.0
**Author**: vskill Contributors
**License**: MIT

## Description

Comprehensive mobile development plugin covering native iOS (SwiftUI), native Android (Jetpack Compose/Kotlin), cross-platform (Flutter, React Native/Expo, Capacitor/Ionic), mobile testing, deep linking, and push notifications.

## Skills

| Skill | Description |
|-------|-------------|
| react-native-expert | React Native and Expo architect and expert: architecture, setup, native modules, performance, and debugging |
| swiftui | iOS native development with SwiftUI, Swift 6, SwiftData, and modern Apple frameworks |
| jetpack-compose | Android native development with Jetpack Compose, Kotlin, Material Design 3, and Hilt |
| flutter | Cross-platform development with Flutter and Dart 3+, Riverpod, GoRouter |
| expo | Expo and React Native New Architecture, EAS Build, Expo Router, and OTA updates |
| mobile-testing | Mobile testing across all platforms: XCTest, Espresso, Detox, Maestro, and device farms |
| deep-linking-push | Deep linking (Universal Links, App Links) and push notifications (APNs, FCM) |
| capacitor-ionic | Web-to-mobile development with Capacitor 6+ and Ionic Framework |
| appstore | App Store Connect automation via `asc` CLI: TestFlight, submissions, metadata, signing, analytics, Xcode Cloud |

## Installation

```bash
vskill add --repo anton-abyzov/vskill --plugin mobile
```

## Requirements

- vskill CLI installed
