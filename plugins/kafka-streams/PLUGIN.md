# Kafka Streams Plugin

**Version**: 1.0.0
**Author**: SpecWeave Contributors
**License**: MIT

## Description

Specialized plugin for Kafka Streams topology design and stream processing. Covers KStream, KTable, and GlobalKTable abstractions, topology patterns, joins, windowing strategies, and exactly-once semantics for building production-grade stream processing applications.

## Skills

| Skill | Description |
|-------|-------------|
| kafka-streams-topology | Kafka Streams topology design expert covering KStream vs KTable vs GlobalKTable, topology patterns, stream operations, joins, windowing strategies, and exactly-once semantics |

## Installation

```bash
vskill add specweave --plugin sw-kafka-streams
```

## Requirements

- SpecWeave core plugin (sw@specweave)
- Apache Kafka Streams knowledge
- Java or Kotlin development environment
