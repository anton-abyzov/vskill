# Frontend Plugin

**Version**: 1.0.0
**Author**: SpecWeave Contributors
**License**: MIT

## Description

Complete frontend development ecosystem for React, Next.js, Vue, and Angular. Provides expert guidance on component architecture, design systems, state management, performance optimization, and polished UI implementation. Includes deep codebase analysis capabilities, design token management, and Figma integration via the official Figma MCP server.

## Skills

| Skill | Description |
|-------|-------------|
| frontend | Expert frontend developer for React, Vue, Angular with hooks, state management, form handling, and accessibility best practices |
| nextjs | Next.js 14+ App Router specialist with Server Components, SSR/SSG, streaming, caching strategies, and deployment optimization |
| frontend-architect | Frontend architecture expert for React, Next.js, Vue, Angular with Atomic Design, state management patterns, and routing strategies |
| design-system-architect | Design systems expert with Atomic Design, design tokens, theming infrastructure, multi-brand support, and Storybook-driven development |
| frontend-design | UI/UX design expert for polished, production-ready interfaces with micro-interactions, animations, and distinctive visual characteristics |
| figma | Figma design-to-code expert with MCP integration for implementing designs, extracting tokens, managing Code Connect, and responsive component generation |
| code-explorer | Deep codebase analysis for tracing feature implementations, understanding data flow, and mapping module dependencies |

## Installation

```bash
vskill add specweave --plugin sw-frontend
```

## Requirements

- SpecWeave core plugin (sw@specweave)
