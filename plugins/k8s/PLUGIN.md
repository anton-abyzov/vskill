# Kubernetes Plugin

**Version**: 1.0.0
**Author**: vskill Contributors
**License**: MIT

## Description

Comprehensive Kubernetes orchestration plugin for manifest generation, Helm templating, GitOps deployment automation, and security policies. Covers production-grade cluster configuration, declarative infrastructure management, and cloud-native best practices.

## Skills

| Skill | Description |
|-------|-------------|
| gitops-workflow | Implement GitOps workflows with ArgoCD and Flux for automated, declarative Kubernetes deployments with continuous reconciliation |
| helm-chart-scaffolding | Design, organize, and manage Helm charts for templating and packaging Kubernetes applications with reusable configurations |
| k8s-manifest-generator | Create production-ready Kubernetes manifests for Deployments, Services, ConfigMaps, and Secrets following best practices |
| k8s-security-policies | Implement Kubernetes security policies including NetworkPolicy, PodSecurityPolicy, and RBAC for production-grade security |

## Installation

```bash
vskill add --repo anton-abyzov/vskill --plugin k8s
```

## Requirements

- vskill CLI installed
- Kubernetes cluster access
- kubectl CLI installed
