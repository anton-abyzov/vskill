# Cloud Cost Optimization Plugin

**Version**: 1.0.0
**Author**: SpecWeave Contributors
**License**: MIT

## Description

FinOps expertise for optimizing cloud costs across AWS, Azure, and GCP. Provides current 2025 pricing data, right-sizing recommendations, reserved instance strategies, spot instance optimization, and cost allocation best practices. Helps teams reduce cloud spend by 20-60% through systematic optimization.

## Skills

| Skill | Description |
|-------|-------------|
| aws-cost-expert | AWS cost optimization expertise - EC2 Reserved Instances, Savings Plans, Spot, Lambda optimization, Cost Explorer, Trusted Advisor |
| cloud-pricing | Expert knowledge of cloud provider pricing models for AWS, Azure, GCP covering compute, storage, database, networking, and serverless services |
| cost-optimization | FinOps expert for AWS/Azure/GCP cost optimization - right-sizing, reserved instances, savings plans, spot instances |

## Commands

| Command | Description |
|---------|-------------|
| (Integrated skills - no separate commands) | Skills activate automatically based on context and keywords |

## Installation

```bash
vskill add specweave --plugin sw-cost-optimizer
```

## Requirements

- SpecWeave core plugin (sw@specweave)
- Access to cloud provider cost dashboards (Cost Explorer, Billing reports, etc.)
